Description

AbstractRecreation sites dataset describes recreation sites (such as picnic and camping sites) within State Forest. This dataset provides valuable information to promote these sites for public use as well as assisting staff in their management of these sites. All recreation sites within State Forest have been captured and recorded with a Trimble Pro XR GPS and are actively promoted to the public and maintained by the Department of Environment, Land, Water and Planning.

Data Quality

LineageData Set Source:
Primary dataset based on GPS field work.

The location and contents were derived from the RECWEB_FAC_RECREATION_SITE table. Other tables with a RECWEB_ACT prefix are joined to the Recreation Site using the ALLOWED_AT and SERIAL_NO fields respectively. This tells which activities are allowed in a Recreation Site location.
Collection Method:
Trimble pro XR GPS with external atenna.
Processing Steps:
Location:
All data has been post processed by base stations. A customised processing system screens the data to ensure all serial numbers are unique and to detect spelling errors.

Data Creation:
Facility and activity tables are dumped from the recweb database into CGDLDEV 'FORESTS' schema. These tables undergo a QA to remove and report back items that dont have a location or SERIAL_NO. Locations (X, Y values) are used to create the points. SERIAL_NO field is used to join associated tables.
Positional AccuracyData has been collected with a Trimble pro XR GSP with an external antenna. The majority of points have sub metre accuracy, detailed in the attributes. The upper limit of accuracy is within 10m for 95% of the data. All data has been post processed using GPSnet base stations.
Attribute AccuracyRecreation Facility attributes were collected on site. The majority of the attributes are from drop down lists which minimises errors.
Logical ConsistencyThe majority of points have sub metre accuracy. All data has been post processed using GPSnet base stations.
