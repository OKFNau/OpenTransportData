Description

AbstractRecreation Tracks within State Forest dataset endeavors to describe all tracks within State Forest. This dataset will assist staff in their management roles and facilitate promotion to the public. All recreation track data within State Forest have been captured and recorded with a Trimble Pro XR GPS. All tracks are actively promoted to the public and maintained by the Department of Environment, Land, Water and Planning. This dataset was initially created as part of the 2004 Recreation Facilities Database project and is actively maintained and updated to reflect changes to our recreationals tracks (including walking, mountain biking and other tracks).

Data Quality

LineageData Set Source:
Primary dataset based on GPS field work.
Collection Method:
Trimble pro XR GPS with external atenna.
Processing Steps:
All data has been post processed by base stations. A customised processing system screens the data to ensure all serial numbers are unique and to detect spelling errors.
Positional AccuracyData has been collected with a Trimble pro XR GSP with an external antenna. The majority of points have sub metre accuracy, detailed in the attributes. The upper limit of accuracy is within 10m for 95% of the data. All data has bee post processed using GPSnet base stations.
Attribute AccuracyRecreation Facility attributes were collected on site. The majority of the attributes are from drop down lists which minimises errors.
Logical ConsistencyThe majority of points have sub metre accuracy. All data has been post processed using GPSnet base stations.
